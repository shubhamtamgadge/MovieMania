 Movie Booking Application
---------------------------

Developed a console-based Movie Booking Application that allows users to select a city, choose a theater and show,
view available seats, book seats, and receive the total booking price.

Also added user registration and booking cancellation functionalities.

Functionalities - 
- select a city 
- select a theater
- select show
- view available seats 
- book seats 
- receive the total booking price
- user registration 
- cancel booking 

Tech Stack : Java, JDBC, MySQL
